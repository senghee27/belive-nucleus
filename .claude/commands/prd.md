# /prd

Write documentation for a completed feature.

Steps:
1. Ask: what feature was just completed?
2. Ask: summarize what was built in plain language
3. Read the relevant files from the codebase
4. Produce all three documents:
   - docs/features/[feature-name]/product-spec.md
   - docs/features/[feature-name]/technical-spec.md
   - docs/features/[feature-name]/feature-brief.md
5. Commit all three files with message:
   docs: add PRD for [feature-name]
